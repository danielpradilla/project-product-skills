# Skill Index

A browseable index of the Project + Product skill set. Each skill lives in `skills/<skill-name>/SKILL.md`.

## Choose The Workflow

| Skill | Use When |
|---|---|
| `project-product-router` | Choose the right Project + Product skill for a request and sequence multiple skills when the work spans discovery, requirements, delivery, launch, PRINCE2 governance, or team rituals. |

## Run PRINCE2 Projects

| Skill | Use When |
|---|---|
| `prince2-startup-brief` | Prepare the PRINCE2 Starting Up a Project material: mandate review, project brief, outline business case, project product description, initiation stage plan, lessons, and startup decisions. |
| `prince2-initiation-pack` | Build the PRINCE2 Initiating a Project pack: project initiation documentation, business case, project plan, controls, management approaches, product descriptions, and baseline decisions. |
| `prince2-direction-gate-brief` | Prepare PRINCE2 Directing a Project gate briefs for project board decisions, including initiation authorization, stage authorization, exception decisions, and closure authorization. |
| `prince2-stage-control-pack` | Run PRINCE2 Controlling a Stage: work package tracking, highlight reports, risk and issue control, tolerance checks, change handling, and corrective action planning. |
| `prince2-product-delivery-pack` | Prepare PRINCE2 Managing Product Delivery material: work package acceptance, team plans, checkpoint reports, quality evidence, delivery confirmation, and handback notes. |
| `prince2-stage-boundary-pack` | Prepare PRINCE2 Managing a Stage Boundary material: end stage report, next stage plan, updated business case, updated project plan, lessons, risks, issues, and exception plan if needed. |
| `prince2-closure-pack` | Prepare PRINCE2 Closing a Project material: product acceptance, end project report, lessons report, benefits review plan, handover, follow-on actions, and closure recommendation. |
| `prince2-documentation-map` | Map the PRINCE2 management products and project documentation needed by process, including startup, initiation, stage control, product delivery, stage boundary, direction gates, and closure. |
